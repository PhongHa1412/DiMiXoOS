# Đị Mi Xô OS

Một ứng dụng web mô phỏng hệ điều hành giả lập (Fake OS) chạy hoàn toàn trong **một file HTML duy nhất**. Chỉ cần mở file là chạy ngay, không cần backend, không cần build.

## Tính năng chính

- ✅ **Chỉ 1 file `index.html`** chứa đầy đủ HTML + CSS + JavaScript.
- ✅ Giao diện OS desktop: wallpaper, taskbar, start menu, cửa sổ draggable/resizable, minimize/maximize/close.
- ✅ Hệ thống cửa sổ đa nhiệm mượt, có hiệu ứng và toast notification.
- ✅ File system giả lập bằng LocalStorage (file & recycle bin).
- ✅ Fake terminal: `help`, `ls`, `open`, `clear`, `about`.
- ✅ Fake browser hiển thị trang nội bộ.
- ✅ Nhạc nền bật/tắt được + sound effect.
- ✅ Boot screen khi load trang, theme Dark/Neon toggle.
- ✅ Chủ đề brainrot meme Việt: Đị Mi Xô OS + chó mascot + bã mía + khô gà + trà từ tay + ộ i i.

## Cách chạy

1. Mở trực tiếp file `index.html` trong trình duyệt.
2. Không cần cài đặt gì thêm.

## Gợi ý sử dụng

- Nhấn **Start** để mở menu ứng dụng.
- Click icon trên desktop để mở nhanh.
- Dùng app **Meme Terminal** để gõ lệnh.
- App **Bã Mía Files** cho phép tạo/xóa file giả.
- App **Bã Mía Recycle Bin** để khôi phục hoặc xóa sạch.

## Yêu cầu hệ thống

- Trình duyệt desktop hiện đại (Chrome/Edge/Firefox).
- Có thể chạy offline.
